Reinforcement Learning

  1. Reinforcement is the branch of machine learning.
  2. RL in robotics: It focuses on robot learning and making decision through trail and error process.
  3. The RL model provides feeback to the robot based on its action.

Types of RL:

  1. Positive RL: It is defined as when an action is correct and increases reward.
  2. Negative RL: When the wrong action is taking by the robot and decreases the points of reward.

Key components of RL:

  1. Strategy: The strategy a robot adopts tp perform its actions based on the enviroment state.
  2. Reward function (model): It increases or decreases based on feedback given to the robot for action performed.
  3. Value function: It is the numeric representation of the reward that robot can expect.
  4. Model: It is te traning stage of the bobot.

# lets define the is_valid function
def is_valid(pos):
  r,c = pos     # r stand for row and c stand for column
  if r < 0 or r >= maze.shape[0]:
    return False
  if c < 0 or c >= maze.shape[1]:
    return False
  if maze[r,c] == 1:
    return False
  return True

# let define the hoose_action function
def choose_action(state):
  if np.random.rand() < epsilon:
    return np.random.choice(len(actions))
  else:
    return np.argmax(Q[state])

rewards_all_episodes = []

for episode in range(num_episodes):
  state = start
  total_rewards = 0
  done = False

  while not done:
    action_index = choose_action(state)
    action = actions[action_index]

    next_state = (state[0] + action[0], state[1] + action[1])

    if not is_valid(next_state):
      reward = reward_fire
      done = True

    elif next_state == goal:
      reward = reward_goal
      done = True
    else:
      reward = reward_step

      old_value = Q[state][action_index]
      next_max = np.max(Q[next_state]) if is_valid(next_state) else 0

      Q[state][action_index] = old_value + alpha * \
      (reward + gamma * next_max - old_value)

      state = next_state
      total_rewards += reward

global epsilon
epsilon = max(0.01, epsilon * 0.995)

:Conclusion The Q-Learning experiment successfully applied Reinforcement Learning principles to teach a virtual robot how to navigate a difficult $10 \times 10$ maze. The robot learned to find the best path by building an internal knowledge base, called a Q-table, that recorded the anticipated future reward for every possible action in every cell of the maze. Over the course of 5,000 training sessions, the robot employed an $\mathbf{\epsilon}$-greedy approach, initially relying on random exploration to find new paths and later prioritizing the exploitation of its known, high-reward routes. This shift in behavior was managed by gradually lowering the epsilon value from $0.5$ to $0.01$. Crucially, the learning was driven by a reward system that strongly incentivized reaching the goal $(+50)$ and heavily penalized hitting walls or boundaries $(-10)$, while also deducting points for every step taken $(-1)$ to promote the shortest safe route. Ultimately, the training was effective, as demonstrated by the rewards plot, which showed a consistent increase in the robot's performance until it reliably located the maze's end.
